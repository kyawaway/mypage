3. Comparison 1 and 2

3.1. Without $?w$-pull

  1. Using Lemma 2.10 and reduction of nets
a('?c'(L12)). ab('?c'(L0)). y(L4,L8,L3). '?c'(L9,L10). '?c'(L15,L16). tensor(L6,L13,L5). b(L14). {system_ruleset. }, ax{ax_(L13). ax_(L14). }, {+L12. '?c_'(L10). '?c_'(L11). }, {+L9. '?c_'(L7). '?c_'(L8). }, cut{cut_(L4). cut_(L5). }, {+L15. '?c_'(L2). '?c_'(L3). }, {'!'(L17,L6). '?d'(L18,L7). '?w'(L1). '?w'(L2). '?w'(L11). ax{ax_(L17). ax_(L18). }. }, {+L0. '?c_'(L1). '?c_'(L16). }.
  1. Translation of $s[x:=t]$ $=$ $(y)$ $z$
a('?c'(L5)). ab(y(L0,L4)). tensor(L2,L6,L1). b(L7). {system_ruleset. }, ax{ax_(L6). ax_(L7). }, {+L5. '?c_'(L3). '?c_'(L4). }, {'!'(L8,L2). '?d'(L9,L3). ax{ax_(L8). ax_(L9). }. }, cut{cut_(L0). cut_(L1). }.

The two results are not equivalent.

However, by using the fusion rule for ?w and ?c in the direction that generates ?w and ?c from a link, along with the rule that places ?w into a box, we can transform (2) into the form of (1). (Since implementing the rule that generates ?w and ?c from a link would cause ?w and ?c to be generated indefinitely, I have not been able to test it.)

3.2. With $?w$-pull

  1. Using Lemma 2.10 and reduction of nets
a('?c'(L5)). ab(y(L0,L4)). tensor(L2,L6,L1). b(L7). {system_ruleset. }, ax{ax_(L6). ax_(L7). }, {+L5. '?c_'(L3). '?c_'(L4). }, {'!'(L8,L2). '?d'(L9,L3). ax{ax_(L8). ax_(L9). }. }, cut{cut_(L0). cut_(L1). }.
  1. Translation of $s[x:=t]$ $=$ $(y)$ $z$
a('?c'(L5)). ab(y(L0,L4)). tensor(L2,L6,L1). b(L7). {system_ruleset. }, ax{ax_(L6). ax_(L7). }, {+L5. '?c_'(L3). '?c_'(L4). }, {'!'(L8,L2). '?d'(L9,L3). ax{ax_(L8). ax_(L9). }. }, cut{cut_(L0). cut_(L1). }.

The two results are equivalent.

{ 
system_ruleset.
// basic
//// tensor-par

tp@@
tensor(X1,Y1,XY1),par(X2,Y2,XY2),cut{cut_(XY1),cut_(XY2)} 
  :- cut{cut_(X1),cut_(X2)},cut{cut_(Y1),cut_(Y2)}.

//// ax-cut
ac@@
cut{cut_(X),cut_(Y)},ax{ax_(Y),ax_(Z)} 
  :- X=Z.

// box
//// dereliction
d@@
{'!'(X1,X2),$g[X1|*X],@r},cut{cut_(X2),cut_(X3)},'?d'(X4,X3) 
  :- cut{cut_(X1),cut_(X4)},$g[X1|*X],@r.

//// nested
pp@@
{'!'(X1,X2),$g1[X1|*X],@r1},cut{cut_(X2),cut_(X3)},{$g2[X3|*X],@r2}
  :- {{'!'(X1,X2),$g1[X1|*X],@r1},cut{cut_(X2),cut_(X3)},$g2[X3|*X],@r2}.

//// weakening
w@@
{'!'(X1,X2),$g[X1|*X],@r},cut{cut_(X2),cut_(X3)},'?w'(X3)
  :- mell.delete(X1,W),{$g[X1|*X],@r},{'?w'(W)}.

//// contration
c@@
{'!'(X1,X2),$g[X1|*X],@r},cut{cut_(X2),cut_(X3)},'?c'({'?c_'(C1),'?c_'(C2)},X3)
  :- mell.copy(X2,A1,A2,A3,B1,B2,C1,C2),{'!'(X1,X2),$g[X1|*X],@r},
      {'?c'({'?c_'(A1),'?c_'(A2)},A3)},{cut{cut_(B1),cut_(B2)}}.


// The equivalence relation is implemented as bidirectional rules.
//w_pull@@
//{'!'(X1,X2),'?w'(X3),$p[X1|*X],@r}
//  :- {'!'(X1,X2),$p[X1|*X],@r},'?w'(X3).
  
//w_push@@
//{'!'(X1,X2),$p[X1|*X],@r},'?w'(X3)
//  :- {'!'(X1,X2),'?w'(X3),$p[X1|*X],@r}.

w_c_fusion@@
'?c'({'?c_'(C1),'?c_'(C2)},C3),'?w'(C2)
  :- C1=C3
}.
